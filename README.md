# USAGE

Don't forget to import the style and script files into your project.
 <link rel="stylesheet" href="lynxalert.css">
<script src="lynxalert.js"></script>

Then call:

```javascript
// color = currently only 'green' and 'red' colors.
// msg = Your message.
lynxAlert(color,msg);
```
